
# IPL Match Analysis (2008-2020)

## 📌 Project Overview
Analysis of IPL match data from 2008 to 2020 using Microsoft Excel.

## 📊 Dataset
- Source: Kaggle
- Files: matches.csv, deliveries.csv
- Total Matches: 1095

## 🔍 Analysis Performed
1. Team Win Count (2008-2020)
2. Toss Decision vs Match Wins
3. Matches Per City
4. Season Wise Matches

## 🛠️ Tools Used
- Microsoft Excel (Pivot Tables, Charts)

## 💡 Key Insights
- Mumbai Indians won the most matches (144)
- Teams choosing to field after winning toss won more matches
- Delhi hosted the most IPL matches (90)
